# Space Boxes Timer

## Vision

Space Boxes Timer is a futuristic desktop productivity application.

The goal is to make focus sessions feel alive.

Each task is represented as a futuristic space container.

When the user starts working:

- the box wakes up
- starts glowing
- emits particles
- becomes connected to space

When the task finishes:

- the box powers down
- a summary appears
- statistics are updated

The application should encourage Deep Work through beautiful visual feedback rather than gamification.

# Architecture Rules

Business Logic must never manipulate HTML directly.

UI rendering must stay inside ui.js.

Timer must stay inside timer.js.

Storage must stay inside storage.js.

Statistics must stay inside statistics.js.

Modal logic must stay inside modal.js.

Application bootstrap stays inside app.js.

Utilities stay inside utils.js.
